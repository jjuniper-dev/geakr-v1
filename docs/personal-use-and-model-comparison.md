# Personal Use and Model Comparison

GEAkr can be useful outside of work as a personal organization pattern. A user can create a folder for personal research, hobbies, planning, writing, learning, or other non-work activities and use the same structure to organize sources, notes, and outputs.

## Personal use

For personal use, the same privacy principle applies: the user is responsible for choosing what information to place in the folder and what chatbot or assistant to use.

Personal use does not automatically mean private. Privacy depends on the storage location, chatbot provider, connectors, permissions, data-use settings, and any local tools involved.

Do not use a personal GEAkr folder to move work information into unapproved tools or personal accounts.

## Model comparison

GEAkr can also help users compare chatbot behavior across tools. Because the folder structure provides a consistent context, a user can ask different assistants to perform the same task using the same source material.

This can help users observe differences in:

- retrieval behavior
- summarization quality
- instruction following
- citation discipline
- artifact generation
- handling of ambiguity
- reasoning over structured notes and files

## Suggested comparison workflow

1. Create a small GEAkr folder using non-sensitive material.
2. Add the same sources and notes for each test.
3. Ask each chatbot the same question.
4. Compare the answers for usefulness, accuracy, traceability, and fit for purpose.
5. Review each tool's privacy settings and terms before uploading or connecting content.

## Boundary

Model comparison should use non-sensitive test material unless the selected tools, storage locations, connectors, and accounts are approved for the information involved.
