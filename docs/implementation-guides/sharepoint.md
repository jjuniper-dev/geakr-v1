# SharePoint Setup (1-minute)

## Goal
Create a shared GEAkr project folder in SharePoint.

## Steps

1. Go to your SharePoint document library.
2. Create a new folder:
   `GEAkr - Project Name`

3. Inside the folder, create:
   - `sources.txt`
   - `notes.md`
   - `briefing.md`

4. Add links to `sources.txt`.

5. Upload supporting files if needed.

6. If your chatbot supports SharePoint access:
   - Point it to this folder
   - Ask it to use the folder as context

## Notes

- Works best with SharePoint connectors.
- Permissions determine whether the bot can read or write.
- Can be shared across teams.
