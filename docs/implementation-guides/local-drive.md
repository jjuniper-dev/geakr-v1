# Local Drive Setup (1-minute)

## Goal
Create a GEAkr project folder on your computer and use it during a chat session.

## Steps

1. Create a folder on your desktop:
   `GEAkr - Project Name`

2. Inside the folder, create:
   - `sources.txt`
   - `notes.md`
   - `briefing.md`

3. Add links to `sources.txt`.

4. Open your chatbot.

5. Copy the contents of `sources.txt` or relevant files into the chat.

6. Say:
   "Use this as my project context and follow the GEAkr instructions."

## Notes

- This is the simplest setup.
- Files are not automatically read unless pasted or uploaded.
- Works without connectors or permissions.
