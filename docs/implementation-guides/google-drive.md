# Google Drive Setup (1-minute)

## Goal
Create a GEAkr project folder in Google Drive with chatbot access.

## Steps

1. Create a new folder in Google Drive:
   `GEAkr - Project Name`

2. Inside the folder, create:
   - `sources.txt`
   - `notes.md`
   - `briefing.md`

3. Add links to `sources.txt`.

4. If your chatbot has Google Drive access:
   - Select or reference the folder
   - Ask the chatbot to use it as project context

5. If write access is enabled:
   - The chatbot may create or update files in the folder

## Notes

- This is the most flexible setup when connectors are enabled.
- Supports both read and write depending on permissions.
- Good candidate for automation and RAG expansion later.
